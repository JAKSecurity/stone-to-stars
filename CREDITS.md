# Credits

**Stone to Stars** is a personal project by **Jeffrey A. Krueger**.

- **Design, code, and original art:** Jeffrey A. Krueger.
- **Sprites and visual effects:** original to this project, generated by a procedural shape-data plus render-pass pipeline (no third-party sprite assets).
- **Sound effects:** procedurally synthesized at runtime with the Web Audio API (oscillators, filtered noise, ADSR envelopes). No third-party SFX files.
- **Code license:** MIT. See [LICENSE](LICENSE).
